Certificate

Download the registered email certificate by certificate Id.

Request

Request URL: https://api-sbx.portaldeassinaturas.com.br/api/v2/registeredEmail/certificate/{id}

Request parameters
id int Certificate Id

Request headers
Token string Subscription key which provides access to this API. Found in your Profile.

Create

List By Filter

In order to list the registered emails it is necessary to inform the indicated period of the start date and the end date (dd/mm/yy) and also the compayId. By default the offset is equal to 0 and the maximum number of records is equal to 100

Request

Request URL: https://api-sbx.portaldeassinaturas.com.br/api/v2/registeredEmail/ListByFilters?start={start}&end={end}&offset={offset}&max={max}&companyId={compayId}

Request parameters
start string Start Date
end string End Date
offset integer offset
max integer Max Results
companyId integer Company Id

Request headers
Token string Subscription key which provides access to this API. Found in your Profile.

Responses

200 OK

List By Period

In order to list the registered emails it is necessary to inform the indicated period of the start date and the end date (dd/mm/yy). By default the offset is equal to 0 and the maximum number of records is equal to 100

Request

Request URL: https://api-sbx.portaldeassinaturas.com.br/api/v2/registeredEmail/ListByPeriod?inicio={inicio}&fim={fim}&offset={offset}&max={max}

Request parameters
inicio string Start Date
fim string End Date
offset integer Offset
max integer Max Results

Request headers
Token string Subscription key which provides access to this API. Found in your Profile.

Responses

200 OK
