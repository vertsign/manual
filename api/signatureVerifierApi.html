Verify Signature

Allows to check the signatures of any signed document. Verifies signatures in the uploaded file.

Request

Request URL: https://api-sbx.portaldeassinaturas.com.br/api/v2/SignatureVerifier/verifySignature

Request headers
Content-Type (optional) string Media type of the body sent to the API.
Token string Subscription key which provides access to this API. Found in your Profile.

Request body

application/json
